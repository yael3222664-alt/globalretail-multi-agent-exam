# GlobalRetail - Data Processing Task (Practical Exam)

This repository contains the solution for the practical portion of the GlobalRetail technical assessment. 
The project focuses on reading, cleaning, analyzing, and summarizing sales data from a CSV file using **Pure Python** only. It strictly avoids external data manipulation libraries (such as Pandas or NumPy) to demonstrate core Python proficiency, algorithm design, and comprehensive edge-case handling.

## 🗂 Project Files
* `globalretail_task.py` - The main Python script containing the data cleaning, sorting, and profit calculation logic.
* `sales_data.csv` - The raw sales data (input).
* `output.txt` - The final output generated by the script, including the data quality report.

## 🚀 Local Setup & Execution
The code is written in Python 3 and uses only built-in libraries (`csv`, `os`, `collections`). No external dependencies are required.

1. Ensure Python 3 is installed on your machine.
2. Clone this repository:
   ```bash
   git clone [https://github.com/YOUR_USERNAME/globalretail-multi-agent-exam.git](https://github.com/YOUR_USERNAME/globalretail-multi-agent-exam.git)
   ```
3. Navigate to the project directory:
   ```bash
   cd globalretail-multi-agent-exam
   ```
4. Run the script:
   ```bash
   python globalretail_task.py
   ```

## 📊 Features & Output
Upon execution, the script analyzes the dataset, prints the results to the console, and saves them to `output.txt`. The analysis includes:
* Top 5 largest transactions by total amount.
* Top 5 transactions with the highest profit margin.
* Calculated sums and averages grouped by product category.
* Gross Profit ranking for each category.
* **Data Quality Report (Bonus):** Identification of rows with missing values, negative transaction amounts, or unexpected/extra columns (safely managed via `None` keys in `csv.DictReader`).
